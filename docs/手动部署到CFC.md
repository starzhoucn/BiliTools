## 配置

由于百度云函数的编辑器只能看到入口文件，所有配置直接在入口文件中写入即可。

入口文件中有如下代码：

```javascript
var _globalVar = require('./config/globalVar');

_globalVar.TaskConfig.config = null;

var _dailyTask = require('./task/dailyTask');
```

配置方式如下：

```javascript
var _globalVar = require('./config/globalVar');

_globalVar.TaskConfig.config = {
  cookie: '',
  function: {
    liveSignTask: true,
  },
};

var _dailyTask = require('./task/dailyTask');
```
